# Design Notification System

## Requirements

### Functional Requirements

- Send notifications via multiple channels (push, email, SMS)
- User preferences for notification types
- Notification templates
- Delivery tracking and analytics
- Rate limiting per user
- Priority levels (urgent, normal, low)

### Non-Functional Requirements

**Scale:**
- 10 million users
- 100 million notifications per day
- Peak: 10,000 notifications per second

**Performance:**
- Process notification in < 100ms
- Deliver within 1 minute

**Reliability:**
- No notification loss
- At-least-once delivery
- Retry failed deliveries

## Database Schema

```sql
-- User preferences
CREATE TABLE user_preferences (
  user_id BIGINT PRIMARY KEY,
  email_enabled BOOLEAN DEFAULT TRUE,
  push_enabled BOOLEAN DEFAULT TRUE,
  sms_enabled BOOLEAN DEFAULT FALSE,
  quiet_hours_start TIME,
  quiet_hours_end TIME,
  preferences JSON,
  FOREIGN KEY (user_id) REFERENCES users(id)
);

-- Notification templates
CREATE TABLE notification_templates (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(100) UNIQUE NOT NULL,
  channel VARCHAR(20) NOT NULL,
  subject VARCHAR(255),
  body TEXT NOT NULL,
  variables JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Notification log
CREATE TABLE notifications (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  template_id INT NOT NULL,
  channel VARCHAR(20) NOT NULL,
  priority VARCHAR(20) DEFAULT 'normal',
  status VARCHAR(20) DEFAULT 'pending',
  sent_at TIMESTAMP NULL,
  delivered_at TIMESTAMP NULL,
  failed_at TIMESTAMP NULL,
  error_message TEXT,
  metadata JSON,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_user_id (user_id),
  INDEX idx_status (status),
  INDEX idx_created_at (created_at)
);

-- Device tokens for push notifications
CREATE TABLE device_tokens (
  id BIGINT PRIMARY KEY AUTO_INCREMENT,
  user_id BIGINT NOT NULL,
  token VARCHAR(255) NOT NULL,
  platform VARCHAR(20) NOT NULL,
  active BOOLEAN DEFAULT TRUE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY unique_token (token),
  INDEX idx_user_id (user_id)
);
```

## Notification API

```javascript
class NotificationService {
  constructor(kafka, db) {
    this.kafka = kafka;
    this.db = db;
  }
  
  async sendNotification(notification) {
    const {
      userId,
      templateName,
      data,
      priority = 'normal',
      channels = ['push', 'email']
    } = notification;
    
    // 1. Get user preferences
    const preferences = await this.getUserPreferences(userId);
    
    // 2. Filter channels based on preferences
    const enabledChannels = channels.filter(channel => 
      this.isChannelEnabled(channel, preferences)
    );
    
    if (enabledChannels.length === 0) {
      return { sent: false, reason: 'All channels disabled' };
    }
    
    // 3. Check quiet hours
    if (this.isQuietHours(preferences) && priority !== 'urgent') {
      return { sent: false, reason: 'Quiet hours' };
    }
    
    // 4. Check rate limits
    const canSend = await this.checkRateLimit(userId);
    if (!canSend && priority !== 'urgent') {
      return { sent: false, reason: 'Rate limit exceeded' };
    }
    
    // 5. Get template
    const template = await this.getTemplate(templateName);
    
    // 6. Create notification records
    const notificationIds = [];
    for (const channel of enabledChannels) {
      const id = await this.createNotificationRecord(
        userId,
        template.id,
        channel,
        priority,
        data
      );
      notificationIds.push(id);
      
      // 7. Publish to Kafka
      await this.kafka.publish(`notifications.${channel}`, {
        notificationId: id,
        userId,
        template,
        data,
        priority
      });
    }
    
    return { sent: true, notificationIds };
  }
  
  async getUserPreferences(userId) {
    const prefs = await this.db.query(
      'SELECT * FROM user_preferences WHERE user_id = ?',
      [userId]
    );
    
    return prefs[0] || {
      email_enabled: true,
      push_enabled: true,
      sms_enabled: false
    };
  }
  
  isChannelEnabled(channel, preferences) {
    return preferences[`${channel}_enabled`] !== false;
  }
  
  isQuietHours(preferences) {
    if (!preferences.quiet_hours_start || !preferences.quiet_hours_end) {
      return false;
    }
    
    const now = new Date();
    const currentTime = now.getHours() * 60 + now.getMinutes();
    
    const start = this.timeToMinutes(preferences.quiet_hours_start);
    const end = this.timeToMinutes(preferences.quiet_hours_end);
    
    if (start < end) {
      return currentTime >= start && currentTime <= end;
    } else {
      // Quiet hours span midnight
      return currentTime >= start || currentTime <= end;
    }
  }
  
  async checkRateLimit(userId) {
    const key = `rate_limit:${userId}`;
    const count = await redis.incr(key);
    
    if (count === 1) {
      await redis.expire(key, 3600); // 1 hour window
    }
    
    return count <= 100; // Max 100 notifications per hour
  }
  
  async getTemplate(name) {
    const template = await this.db.query(
      'SELECT * FROM notification_templates WHERE name = ?',
      [name]
    );
    
    if (template.length === 0) {
      throw new Error(`Template not found: ${name}`);
    }
    
    return template[0];
  }
  
  async createNotificationRecord(userId, templateId, channel, priority, data) {
    const result = await this.db.query(
      'INSERT INTO notifications (user_id, template_id, channel, priority, metadata) VALUES (?, ?, ?, ?, ?)',
      [userId, templateId, channel, priority, JSON.stringify(data)]
    );
    
    return result.insertId;
  }
}
```

## Channel Workers

### Push Notification Worker

```javascript
const admin = require('firebase-admin');

class PushNotificationWorker {
  constructor() {
    admin.initializeApp({
      credential: admin.credential.cert(serviceAccount)
    });
  }
  
  async processNotification(message) {
    const { notificationId, userId, template, data } = message;
    
    try {
      // Get device tokens
      const tokens = await this.getDeviceTokens(userId);
      
      if (tokens.length === 0) {
        await this.markAsFailed(notificationId, 'No device tokens');
        return;
      }
      
      // Render notification
      const notification = this.renderTemplate(template, data);
      
      // Send to FCM
      const response = await admin.messaging().sendMulticast({
        tokens,
        notification: {
          title: notification.title,
          body: notification.body
        },
        data: notification.data
      });
      
      // Handle results
      await this.handleResults(notificationId, response, tokens);
      
    } catch (error) {
      await this.markAsFailed(notificationId, error.message);
      await this.scheduleRetry(message);
    }
  }
  
  async getDeviceTokens(userId) {
    const result = await db.query(
      'SELECT token FROM device_tokens WHERE user_id = ? AND active = TRUE',
      [userId]
    );
    
    return result.map(r => r.token);
  }
  
  renderTemplate(template, data) {
    let body = template.body;
    
    // Replace variables
    for (const [key, value] of Object.entries(data)) {
      body = body.replace(new RegExp(`{{${key}}}`, 'g'), value);
    }
    
    return {
      title: template.subject,
      body,
      data
    };
  }
  
  async handleResults(notificationId, response, tokens) {
    if (response.successCount > 0) {
      await db.query(
        'UPDATE notifications SET status = ?, sent_at = ?, delivered_at = ? WHERE id = ?',
        ['delivered', new Date(), new Date(), notificationId]
      );
    }
    
    // Handle invalid tokens
    response.responses.forEach((resp, idx) => {
      if (resp.error && resp.error.code === 'messaging/invalid-registration-token') {
        this.deactivateToken(tokens[idx]);
      }
    });
  }
  
  async markAsFailed(notificationId, error) {
    await db.query(
      'UPDATE notifications SET status = ?, failed_at = ?, error_message = ? WHERE id = ?',
      ['failed', new Date(), error, notificationId]
    );
  }
}
```

### Email Worker

```javascript
const nodemailer = require('nodemailer');

class EmailWorker {
  constructor() {
    this.transporter = nodemailer.createTransport({
      host: process.env.SMTP_HOST,
      port: 587,
      secure: false,
      auth: {
        user: process.env.SMTP_USER,
        pass: process.env.SMTP_PASS
      }
    });
  }
  
  async processNotification(message) {
    const { notificationId, userId, template, data } = message;
    
    try {
      // Get user email
      const user = await this.getUser(userId);
      
      // Render email
      const email = this.renderTemplate(template, data);
      
      // Send email
      await this.transporter.sendMail({
        from: process.env.FROM_EMAIL,
        to: user.email,
        subject: email.subject,
        html: email.body
      });
      
      // Mark as sent
      await db.query(
        'UPDATE notifications SET status = ?, sent_at = ? WHERE id = ?',
        ['sent', new Date(), notificationId]
      );
      
    } catch (error) {
      await this.markAsFailed(notificationId, error.message);
      await this.scheduleRetry(message);
    }
  }
  
  renderTemplate(template, data) {
    let subject = template.subject;
    let body = template.body;
    
    // Replace variables
    for (const [key, value] of Object.entries(data)) {
      subject = subject.replace(new RegExp(`{{${key}}}`, 'g'), value);
      body = body.replace(new RegExp(`{{${key}}}`, 'g'), value);
    }
    
    return { subject, body };
  }
}
```

```javascript
const twilio = require('twilio');

class SMSWorker {
  constructor() {
    this.client = twilio(
      process.env.TWILIO_ACCOUNT_SID,
      process.env.TWILIO_AUTH_TOKEN
    );
  }
  
  async processNotification(message) {
    const { notificationId, userId, template, data } = message;
    
    try {
      // Get user phone
      const user = await this.getUser(userId);
      
      if (!user.phone) {
        await this.markAsFailed(notificationId, 'No phone number');
        return;

      // Render SMS
      const sms = this.renderTemplate(template, data);
      
      // Send SMS
      await this.client.messages.create({
        body: sms.body,
        from: process.env.TWILIO_PHONE,
        to: user.phone
      });
      
      // Mark as sent
      await db.query(
        'UPDATE notifications SET status = ?, sent_at = ? WHERE id = ?',
        ['sent', new Date(), notificationId]
      );
      
    } catch (error) {
      await this.markAsFailed(notificationId, error.message);
      await this.scheduleRetry(message);
    }
  }
}
```

## Retry Mechanism

```javascript
class RetryManager {
  async scheduleRetry(message, attempt = 1) {
    if (attempt > 3) {
      // Max retries reached
      await this.markAsPermanentFailure(message.notificationId);
      return;
    }
    
    // Exponential backoff: 1min, 5min, 15min
    const delays = [60000, 300000, 900000];
    const delay = delays[attempt - 1];
    
    setTimeout(async () => {
      await kafka.publish(`notifications.${message.channel}.retry`, {
        ...message,
        attempt: attempt + 1
      });
    }, delay);
  }
  
  async markAsPermanentFailure(notificationId) {
    await db.query(
      'UPDATE notifications SET status = ? WHERE id = ?',
      ['permanent_failure', notificationId]
    );
  }
}
```

## Notification Templates

```javascript
class TemplateManager {
  async createTemplate(template) {
    const { name, channel, subject, body, variables } = template;
    
    await db.query(
      'INSERT INTO notification_templates (name, channel, subject, body, variables) VALUES (?, ?, ?, ?, ?)',
      [name, channel, subject, body, JSON.stringify(variables)]
    );
  }
  
  async renderTemplate(templateName, data) {
    const template = await this.getTemplate(templateName);
    
    let rendered = template.body;
    
    // Replace variables
    for (const variable of template.variables) {
      const value = data[variable] || '';
      rendered = rendered.replace(new RegExp(`{{${variable}}}`, 'g'), value);
    }
    
    return rendered;
  }
}
```

## Analytics

```javascript
class NotificationAnalytics {
  async getStats(startDate, endDate) {
    const stats = await db.query(`
      SELECT 
        channel,
        status,
        COUNT(*) as count,
        AVG(TIMESTAMPDIFF(SECOND, created_at, sent_at)) as avg_delivery_time
      FROM notifications
      WHERE created_at BETWEEN ? AND ?
      GROUP BY channel, status
    `, [startDate, endDate]);
    
    return stats;
  }
  
  async getUserStats(userId) {
    const stats = await db.query(`
      SELECT 
        channel,
        COUNT(*) as total,
        SUM(CASE WHEN status = 'delivered' THEN 1 ELSE 0 END) as delivered,
        SUM(CASE WHEN status = 'failed' THEN 1 ELSE 0 END) as failed
      FROM notifications
      WHERE user_id = ?
      GROUP BY channel
    `, [userId]);
    
    return stats;
  }
}
```

## .NET Implementation

```csharp
public class NotificationService
{
    private readonly IKafkaProducer _kafka;
    private readonly IDbConnection _db;
    private readonly IDistributedCache _cache;
    
    public async Task<NotificationResult> SendNotificationAsync(
        long userId,
        string templateName,
        Dictionary<string, string> data,
        string[] channels = null,
        string priority = "normal")
    {
        channels ??= new[] { "push", "email" };
        
        // Get user preferences
        var preferences = await GetUserPreferencesAsync(userId);
        
        // Filter enabled channels
        var enabledChannels = channels
            .Where(c => IsChannelEnabled(c, preferences))
            .ToList();
        
        if (!enabledChannels.Any())
        {
            return new NotificationResult
            {
                Sent = false,
                Reason = "All channels disabled"
            };
        }
        
        // Check rate limit
        if (!await CheckRateLimitAsync(userId) && priority != "urgent")
        {
            return new NotificationResult
            {
                Sent = false,
                Reason = "Rate limit exceeded"
            };
        }
        
        // Get template
        var template = await GetTemplateAsync(templateName);
        
        // Send to each channel
        var notificationIds = new List<long>();
        foreach (var channel in enabledChannels)
        {
            var id = await CreateNotificationRecordAsync(
                userId, template.Id, channel, priority, data);
            
            notificationIds.Add(id);
            
            await _kafka.ProduceAsync($"notifications.{channel}", new
            {
                NotificationId = id,
                UserId = userId,
                Template = template,
                Data = data,
                Priority = priority
            });
        }
        
        return new NotificationResult
        {
            Sent = true,
            NotificationIds = notificationIds
        };
    }
}
```

## Best Practices

- **User preferences** - Respect user notification settings
- **Rate limiting** - Prevent notification spam
- **Priority levels** - Urgent notifications bypass quiet hours
- **Retry with backoff** - Handle transient failures
- **Template system** - Consistent messaging
- **Analytics** - Track delivery rates
- **Idempotency** - Prevent duplicate notifications
- **Batch processing** - Group notifications when possible

## Interview Tips

- **Start with requirements** - Channels, scale, reliability
- **Design API** - Send notification with preferences
- **Show workers** - Separate workers per channel
- **Implement retry** - Exponential backoff
- **Discuss templates** - Variable substitution
- **Address scale** - Kafka for queuing, workers for processing

## Summary

Notification system sends messages via multiple channels (push, email, SMS). Use Kafka for queuing and separate workers per channel. Respect user preferences and quiet hours. Implement rate limiting (100/hour). Support priority levels for urgent notifications. Use templates with variable substitution. Retry failed deliveries with exponential backoff (1min, 5min, 15min). Track delivery status and analytics. Store device tokens for push notifications. Handle 10K notifications/second. Essential pattern for user engagement and communication.
